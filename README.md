# Brain Tumor Classification using Gene Expression 
# Description
This project predicts brain tumor type using gene expression data.

# Technologies Used
- Python
- Scikit-learn
- PCA
- Random Forest
- Streamlit

# How to Run

1. Train model:
   python train.py

2. Run app:
   streamlit run app.py

# Model
- PCA for feature reduction
- Random Forest for classification
